How it works
📚 Classes

Words are grouped into Classes (Class 1, Class 2, …). Each class is like a lesson or chapter. You can practice one class at a time or mix several together using the class pill on the main screen.

➕ Adding Words

Paste words one per line. Each line must have the word and its translation separated by a dash or arrow:

привет – hello
спасибо – thank you
мир → world

You can paste a whole list at once. Choose an existing class or let it create a new one automatically.

📋 Managing Words

Tap "View all words & manage" to see all your words organised by class.

☑ Checkbox (✓ / ✕) — tap it to exclude a word from all games. Useful for words you already know well. Tap again to include it back.
🗑 Delete a word — hold (long-press) any word row for about 1 second. A delete confirmation will appear.
📁 Delete a class — in the Add Words screen, hold (long-press) any class chip for about 1 second. This deletes the class and all its words.
